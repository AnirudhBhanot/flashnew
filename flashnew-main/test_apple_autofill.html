Expected Behavior

When you select a company:

  1. All form fields should populate instantly
  2. A success message appears briefly at the top
  3. You can continue with the assessment using the pre-filled data
  4. Data persists through all assessment steps

Troubleshooting

If fields aren't populating:

  • Check browser console for errors
  • Ensure you're in development mode
  • Try refreshing the page and selecting again
